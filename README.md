# Virtuecore Ads Scraper

This repository is set up for development in VS Code and GitHub Codespaces.

## Local Usage

1. Open this folder in VS Code.
2. Start coding.

## GitHub Codespaces

1. Push this repository to GitHub.
2. Open the repository on GitHub.
3. Click **Code** -> **Codespaces** -> **Create codespace on main**.

If creation is stuck for more than 5 minutes:

1. Go to https://github.com/codespaces.
2. Stop and delete the stuck codespace for this repo.
3. Create a new codespace from the repository page again.
4. If prompted, choose the smallest machine size first.
